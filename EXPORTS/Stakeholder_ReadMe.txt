[현업 데이터 확인 가이드]

1) 산출물 형태(Output Type)
- 업무 종료 후 손에 쥐어지는 결과물입니다.
- 예: 보고서, 티켓, 결재문서, 발송 로그, 확정표

2) 디지털 발자국(Digital Trace)
- 여러분이 일을 했다는 객관적 증거입니다.
- 시스템이 여러분의 성과를 자동으로 기록해주는 근거입니다.
- 예: 상태값 변경 로그, 결재 승인 로그, 전송/발송 이력, 배치 완료 로그

3) 엑셀 데이터 보는 법
- 한 행 = 하나의 L6 세부업무(독립 산출물 단위)
- 시스템_증거(디지털_발자국) 컬럼은 여러 증거를 | 로 연결해 표시합니다.
